Intoduction to HTML: Attributes

If we want to expand an element's tag, we can do so by adding an attribute.

Attribute Examples

    class: used to select one or more elements for the application of style or logic.

    dir: sets the direction of the text of an HTML element.

    id: used to specify a unique id for an HTML element, which cannot be shared across multiple elements within the same HTML document

    lang: specifies the language of the element's content.

    src: specifies the location of a digital source.

    style: specifies how an element will be rendered in a web browser.

    title: contains and can display the specified information about the element it belongs to.
