Intoduction to HTML: Unordered Lists

In addition to organizing texts into paragraphs, you can also display content in an easy to read list.

You can use an unordered list tag or ul tag to create a list of items in no particular order. An unordered list outlines list items with a bullet point.

The ul element shouldn't hold raw text and won't automatically format text into an unordered list of items. Individual list items must be added as a child element using the list item tag or li. The li tag is used to describe an item in a list.
