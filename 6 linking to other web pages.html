HTML Document Standards: Linking to Other Web Pages

One of the most powerful aspects of HTML (and the internet), is the ability to link to other web pages.

You can add links to a web page by adding an anchor element , an a tag, and including the text of the link in between the opening and closing tags.

An anchor element is incomplete without an href attribute. This attribute stands for hypertext reference and is used to link to a path, or the address to where a file is located(whether it is on your computer or another location).
