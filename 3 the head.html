HTML Document Standards: The Head

So far you've done two things to set up the file properly:

  • Declared to the browser that your code is HTML with the doctype declaration
  • Added the html element, that will contain the rest of your code

The head element contains the metadata for a web page. Metadata is information about the page that isn't displayed directly on the web page.

Unlike the information of the body tag, the metadata in the head is information about the page itself.
